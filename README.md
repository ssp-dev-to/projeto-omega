# 🚀 Projeto Omega

Projeto de integração SSP.

Este projeto tem como objetivo avaliar e estruturar a integração entre membros desenvolvedores em projetos colaborativos.

---

## 📌 Objetivo

Criar uma base sólida de desenvolvimento utilizando Laravel, promovendo:

- Organização de código
- Padronização de commits
- Integração entre desenvolvedores
- Boas práticas de versionamento

---

## 🛠️ Tecnologias Utilizadas

- PHP 8+
- Laravel 12
- MySQL
- Node.js
- Vite

---

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/ssp-dev-to/projeto-omega.git

Entre na pasta:

cd projeto-omega

Instale dependências:

composer install
npm install

Copie o arquivo de ambiente:

cp .env.example .env

Gere a chave da aplicação:

php artisan key:generate

Configure o banco de dados no .env e rode:

php artisan migrate

Inicie o projeto:

php artisan serve
npm run dev
